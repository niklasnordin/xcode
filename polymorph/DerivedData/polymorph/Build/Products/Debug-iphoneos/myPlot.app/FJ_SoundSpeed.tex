\cfbox{white}{
Calculates the speed of sound, $w$, from
\[
w^2 = \frac{RT}{M_w} \frac{c_p}{c_v}\left[ 1 + 2 \delta \left( \frac{\partial \bar{\alpha}}{\partial \delta} \right) + \delta^2 \left( \frac{\partial^2 \bar{\alpha}}{\partial \delta^2} \right) \right]
\]
where $\bar \alpha$ is the residual dimensionless Helmholtz energy (see function FJ\_Rho for more info).\\

\small{Depends on functions: rho, cv and cp}
\vspace{115pt}\\
}

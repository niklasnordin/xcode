The dimensionless Helmholtz free energy for region 3
\[ 
\phi(\delta, \tau) = n_1\ln \delta + \sum_{i=2}^{40}n_i\delta^{I_i}\tau^{J_i}
\]
The function returns specific volume.\\
Since the function depends on the density and not pressure, the coefficients for
the backwards equations for the specific volume $v(p,T)$ needs to be implemented as well.

The dimensionless Gibbs free energy for region 2 in the meta-stable vapour region.
\[ 
\gamma(\pi, \tau) = \gamma^0(\pi, \tau) + \gamma^r(\pi, \tau) 
\]
\[
\gamma^0(\pi, \tau) = \ln \pi + \sum_{i=1}^9 n^0_i \tau^{J_i^0}
\]
\[
\gamma^r(\pi, \tau) = \sum_{i=1}^{13} n_i \pi^{I_i} (\tau-0.5)^{J_i}
\]
The function returns specific volume.\\
